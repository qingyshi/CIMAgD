# CIMAgD: Causally-interpretable Meta-analysis using Aggregate data

<!-- badges: start -->

<!-- badges: end -->

The CIMAgD package is to implement a new method for the
causally-interpretable meta-analysis using aggregate data from trials.

## Installation

You can install the development version of CIMAgD by:

``` r
devtools::install_github("qingyshi/CIMAgD")
```
